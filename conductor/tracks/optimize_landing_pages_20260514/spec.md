# Specification: Optimize landing pages for SEO and conversion

## Objective
Enhance the existing landing pages within the Next.js application to improve Search Engine Optimization (SEO) and increase user conversion rates.

## Scope
- Review and update metadata (title, description, open graph tags) across all main landing pages.
- Ensure semantic HTML structure (proper heading hierarchy, alt tags for images).
- Optimize Call to Action (CTA) placement and visibility.
- Integrate initial performance optimizations to improve Core Web Vitals.

## Out of Scope
- Major redesigns of the overall layout.
- Significant changes to the underlying data architecture or Drizzle ORM schema.
- Adding completely new landing pages that don't currently exist.

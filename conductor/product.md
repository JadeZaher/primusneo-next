# Product Guide

## Project Goal
A web application bootstrapped with create-t3-app using Next.js, tRPC, Tailwind CSS, and Drizzle ORM, serving as a dynamic platform for PrimusNeo.

## Target Audience
The platform caters to a diverse audience:
- **Investors & Crypto Enthusiasts**: Users looking to participate in ICOs, governance, or fractionalization.
- **Tech Enterprises & B2B**: Businesses seeking advanced Web3 infrastructure or consulting services.
- **General Public**: Everyday users interested in exploring the fractal engine and engaging with the 3D oasis.

## Platform Objective
The primary objective of the application is to serve as a **marketing front end, an AI-driven dynamic news tool, and a Sanity-backed hub for developer documentation**. It focuses on driving user acquisition for ICOs, keeping the community informed via automated and curated news/blog posts, and providing a scalable, CMS-managed environment for technical docs.

## Key Features & Priorities
The current development phase prioritizes:
- **Marketing & SEO**: Optimizing landing pages for search engines and conversion.
- **AI-Powered News Engine**: Leveraging OpenRouter and cron jobs to automatically fetch tech news, generate insightful posts, and publish them via Sanity CMS.
- **Sanity-Managed Dev Docs**: Implementing a documentation structure within Sanity CMS to manage technical overviews, API references, and system architecture.
- **User Authentication**: Refining NextAuth integration for secure platform areas.
- **Interactive 3D Elements**: Integrating immersive Spline 3D scenes.
